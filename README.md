# Learning_Log
A web app built with Django that can be used for tracking your progress on any topic you are learning.

## To run this code, follow the instructions using your command prompt:

**Clone the repositry**
```
git clone https://github.com/AyobamiAdebesin/Learning_Log_Web_App.git
cd Learning_Log_Web_App
```

**Create a virtual environment**
```
python -m venv envname
'envname' is the name of the environment
```

**Activate the virtual environment**
```
envname\Scripts\activate
```

**Install the requirements**
```
pip install -r requirement.txt
```
**Run the local server**
```
python manage.py runserver
```
